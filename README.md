## 🛍 Project Title

**Online Shopping System for Korean Skincare Products**  
*(User Registration and Purchasing System)*

---

## ❓ Problem Statement

Many customers find it difficult to access authentic Korean skincare products locally.  
Manual ordering systems are slow and prone to errors.  
There is no centralized platform where customers can browse, compare, and purchase products easily.  

Customers need a secure system for:
- Registration  
- Login  
- Online payment  

---

## 🎯 Project Objectives

The main objectives of this project are:

- To develop an online platform for selling Korean skincare products  
- To allow users to register and securely log in  
- To enable customers to browse products by category (cleanser, toner, serum, sunscreen, etc.)  
- To implement a shopping cart and online purchasing system  
- To allow managers to manage products, orders, and users  

---

## 📦 Scope of the System

This system will:

- ✅ Allow users to register and log in  
- ✅ Display a list of Korean skincare products  
- ✅ Allow users to add products to the cart  
- ✅ Enable online purchasing
